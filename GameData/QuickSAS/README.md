#### What is it?

QuickSAS is a plugin which adds the possibility to have keyboard's shortcuts for the SAS functions.

Keyboard's shortcuts:

* Stability Assist: Keypad 5,
* Prograde: Keypad 8,
* Retrograde: Keypad 2,
* Normal: Keypad 9,
* Anti-Normal: Keypad 3,
* Radial: Keypad 6,
* Anti-Radial: Keypad 4,
* Target: Keypad 7,
* Anti-Target: Keypad 1,
* Maneuver: Keypad 0,
* WarpToNode: Keypad Enter.

#### How to install it?

Unzip all files. Put the QuickSAS folder in your KSP/GameData folder.

#### How to update it?

Unzip all files. Merge the new QuickSAS folder with the old folder which is in your KSP/GameData folder.

#### How to uninstall it?

Delete the QuickSAS folder in your KSP/GameData folder.
